### flutter package 开发

创建 package 项目

```
$ flutter create --template=package <#project_name#>
```

### flutter 插件 开发

```
$ flutter create --template=plugin <#project_name#>

/// 带包名
$ flutter create --org com.example --template=plugin <#project_name#>
```
